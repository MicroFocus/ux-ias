Dialog

Purpose

Think of this like a conversation with the user. You want to make sure that they hear you, and that they give a response, even if they defer the response via 'X'

Use case

  • Confirm deletion of an object
  • Simple forms (1 or 2 inputs)

Nesting

Never nested, always fixed to the window

Tab support

Everything else is disabled. Tab support should begin with content and end with 'X'

Small Displays

Dimensions

  • min width
  • max width

Button Order

Scrim

  • Close on click
  • Absorb mouse events

Positioning

Not exactly middle. Should be slightly above middle (golden ratio)
