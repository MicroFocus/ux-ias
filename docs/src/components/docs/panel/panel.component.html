Panel

General

The panel is a visual component that acts as a general purpose container. Panels are designed to fit the overall design theme and appear cohesively with other components. Information presented within a panel can be customized and divided as needed.

Purpose

A visual container for displaying information and nested components.

Panel Specifics

Common Patterns
  • Panels are used as general purpose containers.
  • Most elements and components can be used within a panel.
  • Information within a panel creates a visual grouping that denotes the elements as being related.
  • Components and elements within a panel can be styled to fit the needs of an application.
  • Panels are often used to divide information on a page into separate sections.

Nesting

Panels accept nearly all other components and elements as nestable items. The functionality of a panel is to display nested elements as a group within a container, separating its content from the rest of the page. Panels themselves will likely be their own element on a page.

Responsive Displays

As screen width decreases, a panel will also shrink as necessary. When a panel narrows its width, it will wrap nested content in a way that maintains the original look as much as possible.
