Icons

Nesting icons in buttons (not .ias-icon-button) should be used sparingly. Use of icons in buttons should be the exception. They should not be used on the left of text in a button. They can be used on the right side of text in a button to display state. If an icon is used alone you should use .ias-icon-button instead.
