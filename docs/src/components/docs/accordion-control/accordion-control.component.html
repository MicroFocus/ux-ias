Accordion Control

General

A container for information that makes use of a short descriptive title bar to define what information is held inside. Information can be shown or hidden from the arrow icon in title bar by the user. This element offers the user the ability to navigate large amounts of content with ease.

Purpose

To hide information behind a descriptive title that informs the user what content is held inside. Proper use of the accordion control will save screen space and result in a clean user interface.

Appearance

Common Patterns

  • Each accordion uses a title bar that has a descriptive title and an action icon that allows for hiding and showing the accordion's content.
  • The content section can implement a number of elements, from dialogue to input fields.
  • Think of each accordion as a vertical tab that organized information in a readable way.

Nestable Components

Content Section

Accordions can make use of a large number of nestable components within their content section. Input fields, dialogue boxes, lists, or any element that is necessary for accomplishing the goal of the accordion can likely be used.
