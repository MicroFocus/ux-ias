Tile Grids

General

Tile grids are the basic layout for a group of tiles within a product. To create a horizontal grouping of tiles use .ias-grid. When creating a vertical tile grouping use .ias-grid and .ias-stacked.

Purpose

Create a group of tiles.

Tile Grid Specifics

Implementation
  • Horizontally grouped tiles (.ias-grid) is the default grouping.
  • Vertically grouped tiles requires both .ias-grid and .ias-stacked.

Responsive Displays

As the screen width decreases a horizontal group of tiles will begin to row wrap elements to a horizontal row below the original row.
