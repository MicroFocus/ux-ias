Tiles

General

Tiles are a navigation component that allow a user to access pages, applications, and other data within a product. At minimum, a tile should possess a recognizable avatar or icon image. Tiles may also be given titles, descriptions, and a favorite icon depending on the size of the tile.

Purpose

A navigational component that can be grouped and has recognizable elements.

Appearance

Tile Specifics

Implementation
  • Every tile will contain at least an avatar or icon image.
  • Tiny Tiles will only have the avatar and icon image.
  • Small Tiles add a title to the tile
  • Medium Tiles will have a title as well as a brief description.
  • Large Tiles increase the size of the icon and avatar images, but still contain the title and description of the medium variation.

Nesting

Nestable Components
  • Avatars .ias-avatar
  • Headings <h3>
  • Icons .ias-icon
  • Paragraphs <p>

Responsive Displays

Horizontally grouped tiles such as in the .ias-tiny example will row wrap as screen width decreases.
