Icon

General

Icons visually add continuity, branding, and flow to otherwise homogeneous data. The contrast added to the UI can help make important data stand out. However, adding too many icons increases visual turbulence and will decrease the icons' value.

Purpose

To create visual continuity for common components and data across a product.

Common Patterns
