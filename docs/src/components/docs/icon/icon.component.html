Icon

Purpose

Visually add continuity, branding, and flow to otherwise homogenous data. The contrast added to the UI can help make important data stand out. Adding too many icons increases visual turbulence and will decrease the icons' value.

The following are three standard sizes. Exceptions will exist, but try to use these sizes, if possible, to contribute to a consistent look.

.mf-small(25x25 pixels)

.mf-medium(50x50 pixels)

.mf-large(75x75 pixels)

Use Case

  • Express state visually
  • Present information w/o text (use icons that have universal meaning in this case)
  • Represent informtion combined with icons
  • Express status
  • Identification
  • Represent actions with optimal space

Icon Font

The top navigation points to the icon font that you can include within your project. The font will give you many common icons and can be colored and sized simply via css.
