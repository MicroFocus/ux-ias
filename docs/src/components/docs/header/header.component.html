Headers

General

Headers are a simple means of adding organization to a page. By using varying sizes and weights of text through header elements, developers can create visually recognizable sections. These sections start with the heaviest visual element (header) followed by smaller sized content.

Purpose

Headers are a textual method of organizing content on a page.

Header Specifics

Implementation
  • <h2> elements are the main headings used on page content.
  • When creating sub-headings for content, use headings smaller than the section heading.
  • Limit the use of the <h1> element to special instances, as it draws too much attention from content.
Common Patterns
  • Use header elements to create different sections of a page.
  • Try to limit the use of headings to three different sizes per page.
  • Headings are primarily used by readers as a scanning tool to navigate to desired page content.

Nesting

Nesting Headers in other Elements

Headers are primarily used within elements such as pages, panels, and dialogs; but, they can be nested in many different places. Headers may also be nested as subsections of other headers to reflect the organization of the content.
