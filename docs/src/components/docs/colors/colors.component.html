Colors

General

Micro Focus makes use of a unified color palette to create a consistent brand image across its products. The primary color palette is detailed below along with sets of individual colors. Each color is coupled with its variable name as well as its color code.

Purpose

Establish consistent and unified color usage throughout the UI and include standard corporate colors from the Micro Focus branding guidelines.

Examples

Micro Focus Primary Color Palette

This is the Micro Focus primary color palette. The Identity and Access product group will use the two highlighted colors as their primary product colors. The decision to use these colors from the primary color palette, rather than the MF Primary blue, are as follows:

  • These colors are close to the pre-MF Identity and Access branding colors.
  • These colors are harmonious when used within UI elements.
  • Feedback from developers and customers that the MF primary blue is too bright.
