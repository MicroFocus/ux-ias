Colors

Establish consistent and unified color usage throughout the UI and include standard corporate colors from the Micro Focus branding guidelines.

Micro Focus Primary Color Palette

This is the Micro Focus primary color palette. The Identity and Access product group will use the highlighted two colors as our primary product colors. The decision to use these colors from the Primary color palette, rather than the MF Primary blue, are as follows:

  • These colors are close to the pre-MF Identity and Access branding colors.
  • These colors are harmonious when used within UI elements.
  • Feedback from developers and customers that the MF primary blue is too bright.
