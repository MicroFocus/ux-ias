Datepicker

General

A datepicker is an interactive calendar overlay that allows for user to easily input a desired date. Datepickers are tied to a standard input field that takes input from the interactive calendar. A user can click into a similar month picker by clicking the month/ year in the datepicker title bar.

Purpose

Allow a user to choose a date from a familiar calendar representation.

Examples

Closed Datepicker

06/29/2017
  • The initial states for a datepicker are .ias-icon-button, .ias-link-icon, or .ias-icon-text-button.
  • A developer can choose which closed datepicker style fits the UI best.
  • The user will open the datepicker overlay by clicking the initial state given here.
  • Closed datepickers will change color when hovered over to indicate interactability.

Appearance

Common Patterns

  • Datepickers are always laid out as a familiar calendar with interactive dates.
  • A month picker element can be accessed by clicking the month/ year in the title of a datepicker.
  • Clicking a month within a month picker steps back to that month and year in a datepicker to allow for a more specific selection.
  • If it is necessary to choose a distant date, consider using a simple input field that uses a format hint instead.
