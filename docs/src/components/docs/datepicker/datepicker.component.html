Datepicker

General

A datepicker is an interactive calendar overlay that allows for user to easily input a desired date. Datepickers are tied to a standard input field that takes input from the interactive calendar. A user can click into a similar month picker by clicking the month/ year in the datepicker title bar.

Purpose

Allow a user to choose a date from a familiar calendar representation.

Appearance

Datepicker Specifics

Implementation
  • Datepickers are displayed in a familiar calendar layout with interactive dates and changeable months.
  • Hovering over calendar elements will highlight those elements to communicate interactivity to the user.
  • Dates outside of the range of the current month should be greyed-out with the class .ias-soften.
  • A month picker can be opened by clicking the month and year in the title of a datepicker.
  • A month picker allows for the user to choose a month in a given year, and change the year with arrow icons.
  • Clicking a month within a month picker steps back to that month and year in the datepicker.
Common Patterns
  • If it is necessary to choose a distant date, consider using a input field with a format hint instead.

Nesting

Nestable Components
  • Icon Button .ias-icon-button
  • Spans <span>
  • Tables
  • Text Icon Button .ias-icon-text-button
Nesting Datepickers in Other Components

Date pickers are a special form of an input field and can be nested wherever you would find an input field. Date pickers will overlay content when opened.

Responsive Displays

Date pickers are not responsive to changing screen width. If a screens width is less than the width of the calendar the calendar will be cut off.
