Form Field

General

Form fields are used to both request desired information from a user as well as be a place to receive that input. When requesting an input from a user, make sure it is clear what is being requested. This can be gleaned from the context of the input field, but more often it will be from an input label or placeholder. If the input format needs to be specific, create a visible hint that expresses the format to the user through a placeholder.

Purpose

Allows a user to enter requested information into an application.
