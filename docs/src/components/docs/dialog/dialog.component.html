Dialog

Purpose

A conversation with the user. Make sure the user can give a response, even if they defer the response by closing the dialog.

Use case

  • View and edit object details
  • Confirm object deletion
  • Simple forms (1 or 2 inputs)

Specifications

  • Dialog glass covers all other UI elements when dialog is displayed.
  • Close button is always an available option.
  • Dialog is moveable so the user can read (through the glass) context underneath.
