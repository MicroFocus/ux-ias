Menus

General

Menus are a toggleable component that provide the user with navigation, actions, options and information. Content within a menu larger than three elements could benefit from grouping elements in logical sections.

Purpose

A list of actionable items that can be shown or hidden with a toggle.

Appearance

Common Patterns

  • Menus overlay other page content when toggled to an open state.
  • Menus provide a cohesive list of items that can be used to navigate a product.
  • Menus also provide actions within a page, such as creating a new comments, changing a display type, etc.
  • A complex menu is a combination different menus within a single menu.
  • When creating a menu with more than three items, consider grouping related elements together and adding a dividing line between sections.
  • Menus can also make use of headings to describe separate sections of elements.
  • Menus can also display information to the user, however large amounts of information is not appropriate.

Nesting

Nesting Menus in Other Components

Menus are paired with a menu toggle button. When a menu toggle button is nested into another component, such as the Nav Bar or comments, the menu associated with the toggle will also been nested.

Nestable Components
  • Buttons .ias-button
  • Lists .ias-list
  • Avatars .ias-avatar
  • Headings .ias-menu-header, .ias-menu-footer, .ias-list-header
