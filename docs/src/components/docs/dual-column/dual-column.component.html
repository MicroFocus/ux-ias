Dual Column

Dual column selection allows the user to choose among several items and order them as he or she prefers via drag and drop. Clicking the checkmark and close icons on an item selects and de-selects the item. Dragging a selected item upward or downward rearranges its order in the selected item list.

General Use

Selected Columns

  • Request
  • Task
  • Assigned to
  • Recipient
  • Due date
  • Claimed
  • Request Date

Available Columns

  • Type
  • Priority
