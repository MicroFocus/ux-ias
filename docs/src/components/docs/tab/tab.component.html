Tabs

General

A tab set is a navigational component that exists within a page and hides or displays content specific to each tab. Tab sets are usually tied to a container that displays the information associated with individual tabs. Clicking a tab will set it as activate and display its information in the associated container. Tabs work much like a horizontal accordion control, however it can only have one active element at a time.

Purpose

A navigational component used to hide or display content of a page.

Tab Specifics

Implementations
  • A selected tab will have the class .ias-active.
  • Disabled tabs will have the class .ias-disabled, resulting in the greyed out, un-selectable tab.
  • Tab sets can include links and icon buttons to add more functionality.
Common Patterns
  • Tab sets show and hide content within a page, allowing users to navigate separate sets of information.

Nesting

Nestable Components
  • Icon Buttons .ias-icon-button

Nesting Tabs in Other Components

Tabs are a navigational component and can be nested within a page where navigation of different sets of data would be useful. Tabs navigational capabilities are restricted to a single page and the content of that page.

Responsive Displays

As the display width becomes smaller, tabs reduce their width and the text within until reaching a minimum width. Text reduced in this way reduces the displayed text and append ellipses at the end. Links and icons will remain on the right side of the tab set as much as possible. If a tab set's minimum width, (all tabs are reduce to minimum size) is larger than the display width they will begin to be cut off.
