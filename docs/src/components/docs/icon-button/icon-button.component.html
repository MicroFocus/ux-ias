Icon Button

Perform an action within the context of a UI page or element.

Default

.ias-icon-button

Disabled

[disabled], .ias-disabled

Special Icon Button

.ias-cta

Icon/Action assignments

Icon Primary Action Other Actions
Check validation Confirm validation
Clear input field Remove tag or from list
Add tags Add to list

Use Cases

  • Perform an action
  • Configuration
  • Use when space is limited or simplicity is desired
  • Menu overflow (hamburger or vertical dots)
  • List item primary and secondary actions

Dimensions

  • 25x25 pixels is the standard size
  • 28x28 pixels is the size when used in the page title component

Accessibility

Always include a [title] attribute.
