Icon Component

Purpose

Visually add continuity, branding, and flow to otherwise homogenous data. The contrast added to the UI can help make important data stand out. Adding too many icons increases visual turbulence and will render the icon not valuable.

Use Case

  • Express state visually
  • Present information w/o text (use icons that have univaersal meaning in this case)
  • Express status
  • Identification
  • Represent actions with optimal space
